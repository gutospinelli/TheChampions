# TheChampions
Skeleton for a Football Manager Game, inspired by The Champions League and Elifoot

---

## PODS

#### This project uses the following PODS - See PODFILE
  - pod 'ChameleonFramework/Swift'  - beautiful framework with amazing colors
  - pod 'RealmSwift' - mobile database for apps
  - pod 'SCLAlertView' - simple alerts and popups
  - pod 'Charts' - easy to use charts in apps and games
  

## Project Organization

XCODE Default Project Organization

    ├── TheChampions.xcodeproj
    ├── TheChampions.xcworkspace          
    ├── README.md                 <- The top-level README for developers using this project.
    ├── TheChampions
    │   ├── Assets.xcassets       <- Images and assets used in this project.
    │   ├── BaseDados.swift       <- Realm Database.
    │   ├── clubes.json           <- JSON that defines player attributes and special moves
    │   ├── Sulfix_VC.swift       <- UI View Controllers
    │   ├── Sulfix_TVC.swift      <- UI Table View Cells
    │   └── Sulfix_Model.swift    <- Model Classes (POCO)
    │
    ├── PODFILE                   <- PODs used in this project are declared here
